# Portofolionew
Sebuah portofolio web modern berperforma tinggi yang dirancang khusus untuk merepresentasikan identitas sebagai Back-end Developer dengan sentuhan estetika visual yang mewah. Proyek ini menggabungkan arsitektur kode yang bersih dengan pengalaman pengguna yang imersif melalui efek glassmorphism dan akselerasi GPU.
